# CLMS → CDSE Migration Status Data

Weekly updated migration status data for the Copernicus Land Monitoring Service (CLMS)
to the Copernicus Data Space Ecosystem (CDSE).

## Contents

- `migration_data.json` — scraped weekly by the automated pipeline.
  Contains the Land Portal portfolio (236 datasets, 24 groups) cross-referenced against
  CDSE services (S3, OData, STAC, OpenEO) with per-group ingestion status.

## Source

Auto-generated by the scraper at `CDSE/Migration_Status/scrape.py`.
